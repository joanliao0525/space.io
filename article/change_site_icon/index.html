How to change site icon ?

with using theme hexo-theme-huweihuang

Posted by Joan Liao on 2019-11-01
Joan's space
  • Home
  • Tags
  • Archives
  • About
如果你使用的是 hexo 部落格，並且套用了 hexo-theme-huweihuang 的主題，
可以參考這篇來將你喜歡的圖案設定成網站圖示喔！

1. 找到檔案：themes/huweihuang/layout/_partial/head.ejs

2. 找一個自己喜歡的 icon ，並放到固定路徑：blog/source/img/

我使用的 icon 是這裡下載的：FLATICON

3. 將 #8 的圖片路徑換成自己喜歡的 icon 路徑

變更前

變更後

4. 儲存變更後再重新整理頁面就可以看到新 icon 囉！
